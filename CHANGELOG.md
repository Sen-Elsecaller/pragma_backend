# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.


## [1.0.0] - 2025-12-25

### Agregado
- Sistema de árbol de decisiones interactivo con 15+ puntos de ramificación por escenario
- Motor de simulación de ambientes académicos universitarios (biblioteca, cafetería, sala de clases)
- Sistema de NPCs con comportamientos adaptativos y diálogos contextuales
- Captura de métricas de desempeño en tiempo real (tiempo de decisión, impacto emocional)
- Sistema de sincronización offline-first con backend
- Gestor de persistencia local con savefiles JSON cifrados
- Cliente HTTP para comunicación con API REST
- Sistema de autenticación con tokens JWT
- Interfaz gráfica completa en Godot Engine 4.5
- Dashboard de progreso del estudiante
- Sistema de retroalimentación post-simulación

### Seguridad
- Implementación de cifrado local para savefiles sensibles
- Validación de tokens JWT antes de sincronización
- Gestión segura de credenciales de usuario

### Técnico
- Desarrollado en Godot Engine 4.5
- Lenguaje principal: GDScript 5.0
- Arquitectura cliente-servidor con capacidad offline completa
